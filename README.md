BRICK GAME 
this is a brick game that can be played on both pc and mobile devices developed using python and pygame.
it includes various levels and  score points to be gained 

how to play 
use mouse to move the smash paddle along the horizzontal axis depending on where the ball could land depending on what you visually see 
the main aim is to break all bricks to move to the next level and ensure you dont loose all lives while doing so
